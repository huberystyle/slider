# slider
一个幻灯片效果的jQuery插件！

使用说明：

```js
$("selector").slider({
  bAutoPlay:true, // 是否自动播放
  interval:4000, // 每张图间隔时间
  bPrevNext:true, // 是否显示 '上一张' 和 '下一张' 按钮
  direction:"upDown", // 图片滚动方向
  bPagenation:true, // 是否相识页面
  pageStyle:"number", // 页面的格式 number or icon
  speed:400 // 滚动的速度
});
```
